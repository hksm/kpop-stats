# kpop-stats

Ongoing project for K-Pop Statistics based on data from Gaonchart.

## Resources
Back-end:
* Java 8
* Maven
* Spring MVC
* Spring Data + Hibernate
* Sring Security
* Spring Boot
* JodaTime
* Cloudinary

Front-end:
* AngularJS
* Twitter Bootstrap
* UI Bootstrap
* tamtakoe's oi.select
* itslenny's angular-bootstrap-file-field
